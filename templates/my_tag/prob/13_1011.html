有趣的排列

题目描述 

       给定一个整数n，表示一共有1到n这n个数字，再给定一个整数m，表示一共有m个询问。然后给定n个数字，依次表示1到n这n个数字的个数，从前往后排列。最后依次给出m个数字，问你第几个位置的数字是什么（从第0个位置开始放）。本题的样例中




	
		

			位置
			0
			1
			2
			3
			4
			5
		

		

			数字
			1
			1
			2
			3
			3
			3
		

	




 

输入格式 

	第一行两个整数n和m（表示n个数字和m个询问）。
	（n<=10000,m<=10000）
	第二到第n+1行表示n个数字的个数（个数<=10）。
	第n+2到n+m+1行表示m个询问。

输出格式 

	m行表示m个询问的结果。

输入样例 

3 5
2
1
3
2
3
4
0
1

输出样例 

2
3
3
1
1
