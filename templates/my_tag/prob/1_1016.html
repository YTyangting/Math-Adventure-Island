OO4-4 矩阵类与向量类的算术运算符重载

题目描述 

﻿请设计向量类Vector3，其中数据成员为x,y,z，均为整数类型。请设计矩阵类Matrix3，其中数据成员为int v[3][3]。 请实现向量与矩阵的乘法运算符的重载；实现矩阵与向量的乘法运算符的重载。

输入格式 

输入两行，第一行是一个向量v的三个整数；第二行是一个矩阵m的9个整数，按行主序排列。

输出格式 

输出四行，第一行是v*v的三个整数值（向量叉积）；第二行是v*m的三个整数值（行向量乘以矩阵）；第三行是m*v的三个整数值（矩阵乘以列向量）；第四行是m*m的九个整数值。

输入样例 

1 2 3
1 0 0 0 2 0 1 2 3

输出样例 

0 0 0
4 10 9
1 4 14
1 0 0 0 4 0 4 10 9
